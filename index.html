프로젝트울산 에너지 자급자족 (US_EMS)
현재 단계원천데이터 수집 계획 수립 중
수집 기간2년 (24개월)
수집 주기15분 단위
목표 발전소15개소

기상 요인(계절·일사량·운량 등)에 따른 발전량 변동성 반영 목적. 1개년 데이터는 사계절 패턴이 1회에 불과해 검증 데이터가 부족하고 학습/검증 분리가 어려움. 2개년 데이터 확보 시 1년차 학습, 2년차 검증(또는 교차 검증)을 통해 예측 모델의 일반화 성능 검증이 가능함.

구름 이동 등으로 인한 10~30분 단위의 단기 출력 변동성 반영 목적. 특히 급격한 변동인 램프 이벤트(Ramp Event)는 1시간 단위 데이터 분석 시 정보가 평활화(유실)됨. 2026년 스위스 국가 연구(6,434개 PV 분석)에서도 램프 이벤트 분석 및 전력시장 기준 해상도로 15분 단위를 제시함.

'2년·15분·15개소' 기준은 사계절 반복성, 단시간 변동성, 발전소 간 다양성을 동시에 확보하기 위한 최소한의 운영형 데이터 수집 기준임.

참고문헌: 스위스 6,434개 PV 발전소 데이터 기반 램프 이벤트 연구 (2026)

태양광 발전량 예측은 단순 분석 과제가 아니라 전력거래소 예측제도와 수익 정산에 직접 연결되는 운영 과제다.

03
태양광 발전 예측이란 — 패턴으로 이해하기

실제 발전 데이터가 어떤 패턴을 갖는지 시각적으로 확인. 아래 차트는 일반적인 태양광 발전 패턴을 표현한 시뮬레이션 예시.

① 하루 발전 패턴 — 맑은 날 vs 흐린 날
날씨에 따라 발전량 차이가 3배 이상 발생. 구름이 핵심 변수.
※ 시뮬레이션 예시 — 실제 발전소 데이터 수집 후 교체 예정
② 예측값 vs 실측값 비교
구름 유입 시간대(10~12시)에서 예측 오차 집중 발생. 목표 오차율 8% 이내.
※ 시뮬레이션 예시 — 목표 오차율 8% 이내 (전력거래소 기준)
③ 일사량(GHI) vs 발전량 상관관계
GHI가 높을수록 발전량 증가. 모델이 학습하는 핵심 패턴. 고온·미세먼지 구간에서 관계 약해짐.
※ 시뮬레이션 예시 — 실제 데이터 수집 후 상관계수 분석 예정
구조 요약: "내일 일사량(GHI)이 얼마일지" 예측(기상 모델) → "그 일사량에서 우리 발전소는 몇 kW 나올지" 계산(발전 모델). 패널 오염·음영·미세먼지는 두 번째 단계에서 오차를 만드는 주요 요인.

데이터 계층 구조

국내 탑 플레이어들이 쓰는 데이터를 정확도 기여도 기준 3계층. 우리는 기본층 완전 확보 + 중간층 일부를 1차 목표로 한다.

기본층 — 모든 회사 공통 · 없으면 시작 불가 · 1차 목표
인버터 실측 발전량
AC/DC 출력, 발전량. 예측 모델의 정답값. 이게 없으면 학습 불가.
기상청 수치예보
일사강도·기온·운량·풍속. 학습용(실측) + 예측용(예보) 둘 다 필요.
설비 고유 정보
용량·경사각·방위각·위경도. 같은 햇빛도 발전소마다 다른 이유.
중간층 — 정확도를 올리는 핵심 · 1차 목표에 포함
현장 센서 (모듈 온도)
모듈 표면 온도 센서. 없으면 외기온도로 추정. 발전량 공식 정확도 직접 향상.
미세먼지 (에어코리아)
PM2.5·PM10 실측+예보. 대기 투과율 저하 → GHI 감소 → 발전량 감소.
고급층 — 95%+ 차별화 · 2차 목표
위성 영상
천리안 2호 실시간 위성. 구름 이동 추적으로 단기 정확도 향상.
패널 이미지 (오염 탐지)
이미지 AI. 오염 위치 71.9%, 유형 분류 98% (국내 연구과제 기준).
