01
왜 태양광 발전량 예측이 필요한가
전력은 생산 = 소비가 실시간으로 맞아야 한다. 화력발전은 켜고 끌 수 있지만, 태양광은 햇빛이 있을 때만 나온다. 태양광이 늘어날수록 전력망 변동성이 커지고, 언제 얼마나 나올지 알아야 나머지 발전과 ESS를 조율할 수 있다.
전력망 안정
발전량 예측으로 나머지 발전소 출력 조절 가능
비용 최적화
ESS 충방전 사전 계획, 외부 전력 구매 최소화
정산 수익
전력거래소 예측제도 — 오차율 8% 이내 정산금

▶ 현재 단계 — A. 데이터 : 어떤 자료가 필요한가

태양광 발전량 예측에 필요한 데이터는 크게 3가지다. 이게 원천데이터의 전부다.

① 발전소가 실제로 얼마나 발전했나
예측 모델의 정답값 (타겟). 시간대별 실제 발전량. 이게 없으면 학습 자체 불가.
② 하늘 상태
일사량, 구름 양, 기온, 풍속, 습도. 과거 실측값(학습용) + 미래 예보값(예측 시 입력) 둘 다 필요.
③ 발전소 고유 특성
설비 용량, 패널 기울기·방향, 위치, 모듈 온도 특성. 정적 데이터 — 한 번만 수집.

데이터 세부 항목 정의

A-1. 발전소 실측 데이터 — 정답값

항목 단위 설명 필수 여부

A-2. 기상 데이터 — 핵심 입력값

학습용 (과거 실측)과 예측용 (미래 예보) 두 종류가 모두 필요하다. 학습은 실측값으로, 실제 예측할 때는 수치예보값을 입력으로 쓴다.
항목 단위 설명 필수 여부
GHI 전천일사량W/m²수평면 전체 일사량 — 제일 중요필수
temp_air 기온°C외기온도 — 모듈 온도 추정에 필요필수
cloud_cover 전운량0~10구름 양 — 일사량 예측 보조필수
wind_speed 풍속m/s패널 냉각 효과권장
humidity 습도%대기 투과율에 영향권장
DNI 직달일사량W/m²직접 태양광 성분선택
DHI 산란일사량W/m²하늘 산란 성분선택

A-3. 설비 고유정보 — 정적 데이터

항목 단위 설명 필수 여부

출력온도계수%/°C보통 -0.4~-0.5, 온도 상승 시 출력 저하권장
어레이 타입—고정식 / 단축추적 / 양축추적권장

05
국내 탑 플레이어 벤치마크

국내 실서비스 기업들이 실제로 쓰는 데이터 구성. 우리가 따라가야 할 기준선.

회사 핵심 데이터 차별점 정확도
해줌 인버터 실측 + 위성 영상 + 수치예보 위성으로 구름 이동 실시간 추적 오차율 최저
대연씨앤아이 RTU 인버터 + 국내외 기상정보 + 주변 발전소 해외 NWP(ECMWF) 보조 활용 95%
식스티헤르츠 위성 영상 + 기상 예보·관측 현장 센서 없이 위치만으로 예측 서비스 제공
동서발전 SCADA + 기상청 Open API 2년 155만 건 학습 데이터 92.7%
공통 결론: 기상 데이터만 쓴 모델보다 설비 데이터(모듈 온도 등)를 추가한 모델이 명확히 더 좋다. 기상 + 설비 두 가지를 합쳐야 한다.
