03
태양광 발전 예측이란 — 패턴으로 이해하기

실제 발전 데이터가 어떤 패턴을 갖는지 시각적으로 확인. 아래 차트는 일반적인 태양광 발전 패턴을 표현한 시뮬레이션 예시.

① 하루 발전 패턴 — 맑은 날 vs 흐린 날
날씨에 따라 발전량 차이가 3배 이상 발생. 구름이 핵심 변수.
※ 시뮬레이션 예시 — 실제 발전소 데이터 수집 후 교체 예정
② 예측값 vs 실측값 비교
구름 유입 시간대(10~12시)에서 예측 오차 집중 발생. 목표 오차율 8% 이내.
※ 시뮬레이션 예시 — 목표 오차율 8% 이내 (전력거래소 기준)
③ 일사량(GHI) vs 발전량 상관관계
GHI가 높을수록 발전량 증가. 모델이 학습하는 핵심 패턴. 고온·미세먼지 구간에서 관계 약해짐.
※ 시뮬레이션 예시 — 실제 데이터 수집 후 상관계수 분석 예정
구조 요약: "내일 일사량(GHI)이 얼마일지" 예측(기상 모델) → "그 일사량에서 우리 발전소는 몇 kW 나올지" 계산(발전 모델). 패널 오염·음영·미세먼지는 두 번째 단계에서 오차를 만드는 주요 요인.

05
데이터 계층 구조

국내 탑 플레이어들이 쓰는 데이터를 정확도 기여도 기준 3계층. 우리는 기본층 완전 확보 + 중간층 일부를 1차 목표로 한다.

기본층 — 모든 회사 공통 · 없으면 시작 불가 · 1차 목표
인버터 실측 발전량
AC/DC 출력, 발전량. 예측 모델의 정답값. 이게 없으면 학습 불가.
기상청 수치예보
일사강도·기온·운량·풍속. 학습용(실측) + 예측용(예보) 둘 다 필요.
설비 고유 정보
용량·경사각·방위각·위경도. 같은 햇빛도 발전소마다 다른 이유.
중간층 — 정확도를 올리는 핵심 · 1차 목표에 포함
현장 센서 (모듈 온도)
모듈 표면 온도 센서. 없으면 외기온도로 추정. 발전량 공식 정확도 직접 향상.
미세먼지 (에어코리아)
PM2.5·PM10 실측+예보. 대기 투과율 저하 → GHI 감소 → 발전량 감소.
고급층 — 95%+ 차별화 · 2차 목표
위성 영상
천리안 2호 실시간 위성. 구름 이동 추적으로 단기 정확도 향상.
패널 이미지 (오염 탐지)
이미지 AI. 오염 위치 71.9%, 유형 분류 98% (국내 연구과제 기준).
해외 NWP (ECMWF)
일사량 포함 48h 예보. 기상청 단기예보 일사량 미제공 보완.

학습 데이터셋 구조

원천데이터 → 전처리·가공 → 학습 데이터셋 3단계.

STEP 1. 원천데이터 수집·적재
7개 테이블 분리 적재. 발전소 인근 시군구 기준 기상 매핑.
발전소 실측 (인버터)
AC/DC 출력 · 발전량 · 모듈온도
15분 단위 수집
기상 데이터 (5종 테이블)
ASOS · NWP · 미세먼지 · 예보
1시간 단위 수집
15분 단위 (인버터)1시간 단위 (기상)총 목표 약 190만 건
STEP 2. 전처리 + 피처 가공
15분→1시간 리샘플링 후 기상 데이터 JOIN. 파생 컬럼 생성.
리샘플링 15min→1h결측치 보간이상치 3σlag_1h · lag_24hrolling_mean_6hsin/cos 인코딩발전량 0~1 스케일링
발전소별 vs 통합 모델: 데이터 충분한 발전소는 맞춤형 모델이 더 정확. 신규 발전소는 0~1 스케일링 후 통합 모델 적용. VPP 운영 시 통합 모델 필수.
STEP 3. 품질 관리 — 학습 데이터 선별
사람 손이 유일하게 가는 단계. 설비이용률 기준 이하 구간 학습 제외.
성능 평가 지표
설비이용률(%) = 계량발전량 ÷ 설비용량 × 100
→ 정상 범위 구간만 학습 데이터로 사용
예측오차율(%) = |예측발전량 − 실제발전량| ÷ 설비용량 × 100
→ 목표: 8% 이내 · 참고: 국내 연구과제 92.7% 정확도

모델 학습 및 예측 — 방법론 정리

데이터 수집 완료 후 진행 예정. 참고용 방법론 정리.

핵심 원칙: 시계열 회귀(regression) 문제. 인버터 실측 발전량이 자동으로 정답값. 모델보다 데이터 품질이 먼저다.
머신러닝 기반
XGBoost · LightGBM
트리 기반 앙상블. 빠르고 정확도 높음. 피처 중요도 직접 확인 가능. 초기 베이스라인으로 먼저 적용.
초기 모델 · 피처 중요도 분석 · 데이터 부족 구간
딥러닝 기반
LSTM
과거 패턴 흐름 기억. "1시간 전이 이랬으면 지금은 이럴 것" 학습.
CNN-LSTM
CNN으로 지역 패턴 추출 후 LSTM으로 흐름 학습. 일사량 변화 포착.
Multi Input Model
실측·기상·설비정보 각각 별도 입력. 데이터 종류 다양한 우리 프로젝트에 적합.
